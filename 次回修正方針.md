# 次回修正方針 - フォーム2行→1行レイアウト変更

## 🎯 目標
Novel AIタブのタイトル・カテゴリ入力を2行から1行レイアウトに変更する

## 📍 現在の状況
- v0.85: 既存のレイアウトのまま
- 前回の修正が反映されていない（間違った場所を編集した可能性）

## 🔍 対象箇所の調査が必要
Novel AIタブ内のフォーム要素を特定する必要がある：

### 探すべき要素
1. **タイトル入力フィールド**
   - 現在: ラベルとinputが縦に配置（2行）
   - 目標: ラベル（左）+ input（右）の横配置（1行）

2. **カテゴリ入力フィールド**  
   - 現在: ラベルとinputが縦に配置（2行）
   - 目標: ラベル（左）+ input（右）の横配置（1行）

### 検索方針
以下のパターンで該当箇所を探す：
- `id="promptTitle"` （既存のタイトル入力）
- Novel AIタブ内のlabel + input構造
- タイトル・カテゴリに関連するフォーム要素

## 💡 実装方針

### CSS/HTMLの変更パターン
```html
<!-- 変更前（2行） -->
<div style="margin-bottom: 8px;">
    <label>タイトル</label>
    <input type="text" id="title-field">
</div>

<!-- 変更後（1行） -->
<div style="margin-bottom: 8px; display: flex; align-items: center; gap: 10px;">
    <label style="min-width: 60px;">タイトル</label>
    <input type="text" id="title-field" style="flex: 1;">
</div>
```

### レイアウト仕様
- **Flexboxレイアウト**: `display: flex; align-items: center;`
- **ラベル幅**: `min-width: 60px`（固定幅）
- **入力欄**: `flex: 1`（残り幅全て使用）
- **間隔**: `gap: 10px`
- **縦マージン**: `margin-bottom: 8px`（維持）

## 🔧 サンプルボタンパネル
検索欄の上に追加する軽量パネル：

```html
<div style="margin-bottom: 8px; background: #f8f9fa; padding: 6px 10px; border-radius: 4px; border: 1px solid #e9ecef;">
    <button onclick="alert('サンプル機能は実装中です')" 
            style="background: #2196F3; color: white; padding: 4px 12px; border: none; border-radius: 4px; font-size: 13px; cursor: pointer;">
        📝 サンプル
    </button>
</div>
```
